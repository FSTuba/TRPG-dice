## 如何執行
在有安裝python的環境下直接執行`trpgdice.py`即可。

## 擲骰子指令說明

### 基本指令：[x]D[y]
  [x]代表骰子數量，[y]代表骰子的面數，舉例`3D6`即"投擲3顆6面骰子的結果相加"。
  程式會忽略空白，並且D可以為大寫或小寫，也支援常數加減如`1D6+2`。

### 建立角色：ch
  這個指令會根據 **克蘇魯的呼喚第六版規則** 產生一組隨機的角色基本數值。

### 結束程式：end
  就結束程式啊，還有什麼好說的？
